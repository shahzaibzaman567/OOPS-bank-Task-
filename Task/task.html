📘 Robot Management System - Task Instructions (Only)

  • آپ نے ایک ایسا system design کرنا ہے جس میں مختلف قسم کے AI Robots create کیے جائیں۔
  • اس project میں آپ کو درج ذیل concepts کو لازمی استعمال کرنا ہے:
    • 🔐 Encapsulation: ہر Robot کی private properties (جیسے ID, Model) encapsulate ہوں گی۔
    • 🧠 Abstraction: Parent class میں abstract-style method ہوگا جیسے getProfile()، جو child classes override کریں گی۔
    • 🧬 Inheritance: Parent class Robot سے child classes inherit کریں گی جیسے CleaningRobot, SecurityRobot وغیرہ۔
    • ⚙️ Generator Function: Robot IDs یا names generate کرنے کے لیے generator function use ہوگا۔
    • 🌐 Async + API: Real robot owner name لینے کے لیے external API سے async fetch ہوگا۔
  • 💻 UI میں کیا ہوگا؟
    • Bootstrap-based ایک card ہوگا جو robot details دکھائے گا۔
    • ایک button ہوگا: Load Next Robot — جس پر click کرنے سے ایک نیا robot create ہو گا۔
    • ہر بار card میں robot کا name, model, type اور assigned user کا نام آئے گا۔
    • 10 robots create ہونے کے بعد button disable ہو جائے گا اور message آئے گا: \"All robots have been loaded.\"
  • 📌 آپ یہ پورا کام ایک single HTML + JS + CSS page میں کرو گے۔
